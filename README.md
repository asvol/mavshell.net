# mavshell.net

Mavlink shell tool, which use [MAVLink.net](https://github.com/asvol/mavlink.net) library.

Example:
```bash
mavshell-net.exe mavlink
```
![listen](img/1.PNG)

```bash
mavshell-net.exe info
```
![listen](img/2.PNG)
```bash
mavshell-net.exe px4info
```
